# Getting Started

To install **RuKit** in your project, simply run the following command in the terminal:

```
npm install rukit
// or
yarn add rukit
```

```tsx
import { Button } from 'rukit';

export default function page() {
  return <Button size="medium">Button</Button>;
}
```
