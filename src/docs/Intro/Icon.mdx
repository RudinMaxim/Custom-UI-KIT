# Icon

The `Icon` component allows you to easily add **SVG** icons to your Rukit components, making your interfaces more visually appealing and user-friendly. It is based on [fonts.google.com/icons](https://fonts.google.com/icons)

```tsx
import { Button } from 'rukit';

export default function page() {
  return <Icon name={search} text={'search'} />;
}
```
