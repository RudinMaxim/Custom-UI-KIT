# Installation

To install **RuKit** in your project, simply run the following command in the terminal:

```cmd
npm install rukit
```

## Peer dependencies

> Please be aware that **react** and **react-dom** are peer dependencies, so make sure to have them installed before proceeding with the installation.

```json
"peerDependencies": {
  "react": "*",
  "react-dom": "*"
}
```
